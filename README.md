# n8n Workflows Repository

This repository contains a collection of **n8n workflows** exported as JSON files.

## 📦 How to Use

Each workflow is provided as a `.json` file. To use them:

1. Open your **n8n** instance.
2. Click on **"Import"** (or paste into the editor).
3. Copy the contents of the desired JSON file from this repository.
4. Paste it into n8n.
5. Save the workflow.

## 🔐 Credentials Setup

After importing a workflow, you will need to configure your own credentials:

* Open each node that requires authentication.
* Replace the existing credential placeholders with your own.
* Test the connection to ensure everything works correctly.

⚠️ **Note:** Credentials are not included for security reasons.

## ⚙️ Requirements

* A running n8n instance (self-hosted or cloud)
* Proper API keys / credentials depending on the workflow integrations
